## ⏱ 如果只读 10 分钟（7 条带走判断）

1. <strong>拆竞品做产品，第一道工序不是抄结构，是查专利</strong>。这台机器人有三个高风险点——头部触摸键、头顶三麦克风、正前方雷达——全程被反复念叨"必须先查专利再开模"，因为开模就是真金白银投进去、没有回头路。
2. <strong>专利规避的标准动作：查到对方权利要求后，把其中一两点做得不一样就行</strong>。圆形改方形、对称改带防呆缺口、整环改 3/4 环——外观稍调就能脱开外观专利。这是消费电子抄改的通用解法。
3. <strong>外观专利能查到、结构/发明专利查不到，不代表安全</strong>——对方很可能用的是"别人的专利"（给它代设计的方案商另行申请）。所以查专利要扩到关联公司，且换关键词多查几轮（一个结构件换个词就可能查不出来）。
4. <strong>软硬结合板（FPC+PCB）是成本和良率的隐形杀手</strong>。竞品身体里用了软硬板、还在软板上贴元器件，导致 PCB 工艺要求高、需要专用治具焊接、组装麻烦。本场共识：能用硬板+标准线材（圆形铜线）连就别用软板。
5. <strong>电机转轴该不该用"模内注塑"是个关键工艺判断</strong>。受扭矩+长期摩擦的金属轴，理想做法是金属件和塑料件一起注塑（模内注塑，强度最好）；但成本和模具难度高。现场提出更省的替代：热熔/加热压入 + 夹具保证垂直度。要不要上模内注塑"后面评估成本再定"。
6. <strong>车载场景必须按夏季 70℃+ 高温设计电池策略</strong>。车内夏天可达七八十度，锂电池在 70℃ 附近"基本是直线"（性能/安全悬崖）、长时间有爆炸风险。所以做<strong>含电池 / 不含电池两个版本</strong>，出海预售几百套时大部分客户要的是不带电池、靠底座 USB 供电的版本。
7. <strong>6mm 空心杯电机功耗极低，USB 2.0（5V/500mA，2.5W）就够驱动整机</strong>——这让"去掉电池、纯底座供电"在工程上成立。

## 📚 深挖：8 个值得单独学的知识点

### 1. 抄改类硬件的"专利规避工作流"（本场真正的主线）
现场反复出现同一套动作：先确定哪几个部位最像竞品（头部触摸、头顶三麦、正前雷达、充电座造型），<strong>开模前必须把这几点的专利查清</strong>。查法有讲究——
- 外观、结构（实用新型）、发明三类分开查；
- 工商信息只查到对方"有外观专利、没查到结构/发明专利"≠安全，<strong>对方可能用的是别人/关联公司申请的专利</strong>，要顺藤查关联公司；
- 同一个结构件<strong>换个关键词可能就查不到</strong>，要换多个词多查几轮；
- 自查 + 找专业外包团队（"做消费级电子的专业检索人员，一两天能大致检索出来"）<strong>并行</strong>，互相补漏。
查到之后的规避动作：<strong>看对方权利要求保护哪几点，把其中一两点做得不一样即可脱开</strong>（圆改方、对称改带防呆缺口、整环改局部）。
<strong>你能用</strong>：任何仿改/对标类产品，把"开模前专利检索"设成硬卡点，列出"必须避开的点"清单再启动结构设计。

### 2. 软硬结合板（FPC+PCB）是成本与良率的隐形杀手
竞品身体内部用了<strong>软硬板，且在软板上还贴了元器件</strong>。代价：PCB 工艺要求高、组装麻烦、需要专用治具才能焊接（不能像普通硬板那样"刮锡膏→贴件→过回流焊"一气呵成）。本场共识——<strong>能拆成几块硬板、中间用标准圆形铜线/排线连，就别沿用它的软硬板</strong>。判断依据是量：模具可重复用、是一次性收费，量上来（月几K～几十K）平摊后单件软板成本可降到几分钱，但前期治具、工艺复杂度的代价实打实。
<strong>你能用</strong>：评估自己产品时，先问"这块软板能不能砍成硬板+标准线材"，软硬板要用得起，得有量来摊成本。

### 3. 受力旋转件的工艺抉择：模内注塑 vs 热熔压入
电机转轴这个位置<strong>既受扭矩又长期摩擦</strong>，纯塑料件日积月累会因应力损坏、强度不够。理想工艺是<strong>模内注塑（in-mold / insert molding）</strong>——把金属件预先塞进模具，和塑胶一起注塑成一体，融合后整体强度最好，但模具难度和成本更高。现场给出更省的替代路径：<strong>对皇冠齿轮这类配合件，用加热热熔把轴压进去 + 做个夹具保证垂直度</strong>，精度要求不高的玩具件足够；难点是垂直度不好保证、易开裂。最终结论是"评估模内注塑会增加多少成本再决定"。
<strong>你能用</strong>：遇到"受力+旋转+塑料"的结构，先判断是否非模内注塑不可；不是精密件的话，热熔压入+夹具往往是性价比更高的方案。

### 4. 车载硬件的高温电池约束 → 双版本策略
车内夏天可达<strong>七八十度</strong>；锂电池在 <strong>70℃ 附近性能"基本是直线"（悬崖式衰减）</strong>，长时间还有爆炸风险。因此产品分<strong>含电池 / 不含电池两个版本</strong>：出海预售几百套里，<strong>大部分客户要的是不带电池版</strong>（怕电池风险），靠底座供电。工程前提是功耗够低——选 <strong>6mm 空心杯电机</strong>、整机能被 <strong>USB 2.0（5V/500mA=2.5W）</strong> 驱动，所以"去电池纯底座供电"成立。注意：电池本身才十来克，但去掉会影响重心，结构上要补配重。
<strong>你能用</strong>：做车载/高温环境电子，把"去电池+外部供电版本"作为默认选项之一；先算清整机功耗能否被标准 USB 口带动。

### 5. 连接器选型：空间够用插拔式、空间紧用上下压合
两种连接器的取舍：<strong>正常插拔式（两片/三片）插拔方便、可靠，但占空间</strong>；<strong>上下压合的超薄型省空间，但牢靠性差、容易松</strong>。原则：空间够优先插拔式；空间不够才上压合超薄型。若必须用很小弯曲半径的软排线，要在结构上做<strong>压板把它压住</strong>，防止松脱（软排线弯曲半径太小容易回弹松动）。
<strong>你能用</strong>：连接器不是越薄越好，先评估装配空间和可靠性，再决定要不要牺牲空间换可靠。

### 6. 板厚与板框：先有"板框图"才能布局
硬件侧原理图已在画，<strong>等结构侧给出"板框（板子外形/尺寸轮廓）"才能布局元器件</strong>。能一块板铺得下就用一块，铺不下再用两块板承接。板厚经验值：<strong>超过 50×50mm 的板最好 1.0mm；二三十毫米的小板可用 0.8mm</strong>；升级部分空间不够最多做到 0.8。这是结构和硬件来回咬合的协作节奏——结构定外形→硬件布板→反馈走线和接口朝向→结构定插座位置。
<strong>你能用</strong>：结构与硬件并行时，明确"板框图"是交接物，先冻结外形再布局，避免反复返工。

### 7. 毫米波雷达：天线搞不了就买模块
正前方那个雷达，<strong>毫米波天线需要专业的人机（天线）设计，团队自认搞不了，所以直接买现成模块</strong>，单独做一块小板扣在结构上。同时澄清一个易混点：顶在外壳上的金属片<strong>不一定是天线，可能是触控感应片（也可能做成 EMC 弹片）</strong>——拆机时别想当然。
<strong>你能用</strong>：射频/天线这类高门槛环节，买成熟模块比自研划算；拆机判断元件功能时多核实，别凭外形猜。

### 8. 充电座"整环 vs 局部环"带来的结构连锁问题
竞品的导光环（ASR 一度误写"导电环"，后澄清是<strong>导光环</strong>）是 <strong>3/4 圈</strong>；本方想做<strong>一整圈</strong>更好看。但整环会让外圈标签件之间"全断开、成孤岛"，失去结构联系，<strong>只能靠中间那根轴 + 底座轴承来支撑</strong>——于是中间的轴要做大、加大轴承托住底盘才不晃。一个外观决定（整环好看）直接牵出结构方案（加粗轴、加大轴承）。
<strong>你能用</strong>：外观和结构是耦合的，任何"为了好看"的改动都要顺着追问"它会不会破坏结构连续性、需要补什么支撑"。

## 🛠 可直接复用的方法（含"不要误读成"）

<strong>方法 1：开模前的专利检索卡点</strong>
- 核心意思：仿改类产品最大的沉没成本风险在专利，必须在开模前用"自查+外包并行、扩查关联公司、多关键词"把高风险点查清。
- 你可以怎么用：把"专利检索通过"设成开模的前置门，列出"必须避开的点"清单。
- 不要误读成：<strong>"只查到外观专利没查到发明/结构专利=安全"</strong>——对方可能借了别人/关联公司的专利。

<strong>方法 2：能砍软板就砍，用硬板+标准线材</strong>
- 核心意思：软硬结合板工艺/良率/组装成本高，能拆成硬板用标准圆形铜线连就别沿用竞品的软硬板。
- 你可以怎么用：评审 BOM 时单独审"软板项"，逐条问能否硬板化。
- 不要误读成：软板一无是处——量足够大时它的模具成本可摊薄，关键是别在没量时为省空间硬上软板。

<strong>方法 3：受力旋转件先问"是否非模内注塑不可"</strong>
- 核心意思：模内注塑强度最好但贵；非精密件常可用热熔压入+夹具替代。
- 你可以怎么用：对每个"受力+旋转"塑料件做一次工艺成本对比，再决定。
- 不要误读成：模内注塑=高级=必须用——成本可能吃掉利润，按精度要求来。
