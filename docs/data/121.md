## 📚 深挖：3 个值得单独学的知识点

### 1. 「合规抢跑反被栽赃」的诉讼复盘——证据决定一切
<strong>是什么</strong>：讲者一方做了个语音小组件（类似"今天早上起床了吗"的互动小组件），先于对方（极客系，疑为安卓侧"语音组件"产品方）合规上线并起量。对方在安卓没做好风控，用户上传了违规语音（现场反复提到"瑟琴的语音"被举报）导致其产品被下架，对方便认定是讲者一方恶意举报/搞竞对，"气势汹汹起诉"。结果："我们律师跟对方律师交换证据之后""法官跟对方律师说你现在最好的结果就是撤诉"，最终开庭前撤诉。
<strong>为什么重要</strong>：这是一个典型的"我先把事做对、对方自己出错却来咬我"的纠纷模型。关键不在谁先上线，而在<strong>证据链</strong>——交换证据后对方律师自知告不赢，法官直接劝撤。它也提醒：抢跑合规者在对方眼里会被解读成"下三滥手段"，舆论/情绪上你未必占理，但法律上证据为王。
<strong>你能用</strong>：被无理起诉时，第一动作是固定/留存证据（上线时间、风控逻辑、沟通记录），交给律师做证据交换，多数"吓唬式起诉"会自行撤诉。

### 2. 播客/音频平台格局现场口径
<strong>是什么</strong>：席间随口盘了一圈——<strong>小宇宙</strong>（被指是"他们家的"即小红书系？现场口径，存疑；"推得好但量没那么大"）、<strong>荔枝</strong>（上市公司，做海外社交，"去年做了 30 多个亿"，但国内业务"天天被打"）、<strong>喜马拉雅</strong>（用户量最大，"被蜻蜓 FM 收了"——此句方向可能反了，通常说法是喜马拉雅体量更大，需核实）。还有一句信号："<strong>小红书要做自己的播客</strong>""越来越多了"。
<strong>为什么重要</strong>：这是从业者茶余饭后的"体感排名"，未必精确，但能快速校准一个赛道里谁大谁小、谁在扩张。"平台要自做内容生态"（小红书做播客）往往是上游动作，会影响下游创作者/工具方。
<strong>你能用</strong>：把"小红书自做播客"记为一个待验证信号；涉及音频分发时，平台体量口径自己再独立核实（本场数字均为饭桌口径）。

### 3. 风口生意的"保质期"：RPA 正被 computer use 干掉
<strong>是什么</strong>：一位讲者自述路径——"22 年我做爬虫，23 年做 RPA，24 年做 AI"，被同桌评"每一波都是风口"。RPA（点点点的自动化，工具点名<strong>影刀、UiBot</strong>）巅峰期"一个月能干十几万"，但现在"被 code 5 的 computer 干的……倍感的体无完肤，完全卖不动了"——因为 computer use"全流程打通、根本不需要学任何东西"。
<strong>为什么重要</strong>：这是一个鲜活的"工具型生意被上一层 AI 能力吞掉"的案例。RPA 的价值是"教机器按固定流程操作"，而通用 computer use 直接让用户用自然语言驱动、无需配置，等于把 RPA 的护城河填平。<strong>任何"帮人省重复操作"的工具，都要警惕被更通用的 agent 替代。</strong>
<strong>你能用</strong>：盘一遍你手里靠"帮人省事/自动化"赚钱的业务，逐条问"通用 AI agent 能不能直接做"；能被替代的，要么往更深的行业 know-how 走，要么趁还能卖的时候转型。

## 🛠 可直接复用的方法（含"不要误读成"）

<strong>方法 1：无理诉讼，先固证再交换证据</strong>
- 核心意思：对方"气势很足"地起诉，多是吓唬；律师交换证据后对方自知告不赢，法官会劝撤诉。
- 你可以怎么用：保留上线时间、风控/合规记录、聊天记录，交律师走证据交换流程。
- 不要误读成：以为先上线就天然有理——决定胜负的是证据，不是先后或情绪。

<strong>方法 2：内容"全平台分发"，不困在要靠人情互推的小生态</strong>
- 核心意思：讲者明确表示不玩极客那种"互相转来转去、抱团互推"，内容"全部分发"。
- 你可以怎么用：把一份内容做成多平台分发，别把增长押在一个要每天"亲亲么么哒"维系关系的小圈子。
- 不要误读成：完全不要关系——而是不要把命脉系在"必须互推才有量"的封闭生态上。

<strong>方法 3：借大场合搭车发布，换 PR 物料</strong>
- 核心意思：一方借"撑场"的活动，顺势发布刚做两周的媒体品牌"逐日前线"，攒一份 PR 物料。
- 你可以怎么用：有新品牌/产品时，找一个现成的大场合搭车首发，低成本拿到曝光和素材。
- 不要误读成：为发布硬办一场活动——搭车别人的场更省成本。
