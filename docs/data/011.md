## 📚 深挖：6 个值得单独学的知识点

### 1. "信任度 > 专业度"——to-小B 服务生意的成交第一性
主角讲得最透的一点："客户的信任度会大于你的专业度，你要忽悠得了客户、又能跟客户（处好）……因为很多都是创业老板、小 B 端、公司中层，要能让他们信任、把项目交给你，这大于你的专业度。"
- <strong>为什么重要</strong>：技术服务的买家其实无法判断你技术好坏，他只能判断"我信不信你能交付"。所以决定成交的是信任信号，不是技术参数。这条几乎适用于一切卖给非专业买家的定制服务（建站、AI 代做、咨询、外包）。
- <strong>你能用</strong>：把获客资源的一部分从"展示专业"转向"建立信任"——案例、面谈、实物、第三方背书。

### 2. 暴利=信息差红利，"说出去就没了"
他坦白：自己"很摆烂地做"也快 100 万，因为"身材（赛道）没看到有人做"；又反复懊悔"单片机是我他妈说出去的，然后好多人做""我一说出去，后面就好多人在做"。还有"4月最好做、十几万的单当天打五六万定金，现在价格低了、做的人多了"。
- <strong>为什么重要</strong>：这是红利型生意的铁律——利润来自信息不对称，公开即衰减。它解释了他为什么一边招人合作、一边又强调"闷声"。
- <strong>你能用</strong>：识别你手上的钱是"技术护城河"还是"信息差红利"。若是后者，节奏上要"早、快、低调"，并提前想好红利消失后的下一站（他选了出海）。

### 3. 线下面谈 + 实物道具 = 信任的物理放大器
他几乎所有大单都是"把客户喊到公司、当面吹牛"成的，线下签了六七单；并用机器狗/机器人现场演示增信——"机器狗都能做，你这些小硬件不是简简单单随便做"。
- <strong>为什么重要</strong>：信任在线下、有实物时传递效率远高于线上聊天。一个"显然很难、而你已经做出来"的实物，能瞬间压掉客户对"你到底行不行"的疑虑，并支撑溢价。
- <strong>你能用</strong>：哪怕你做软件/虚拟服务，也设计一个"线下/可演示/有实感"的环节当成交临门一脚。

### 4. 报价锚定 + 回扣破局：他的两件谈单武器
- <strong>报价锚定</strong>：基础一整套约 3 万成本，他常报 4–5 万；并直接对客户说"我们比市面贵，但更靠谱"，把高价转成信任信号。
- <strong>回扣破局</strong>：对甲方一线负责人，他"很直白就说，您是负责人，项目拿下来给你回扣"，自称"这个点我能拿下很多单"。
- <strong>为什么重要</strong>：前者是定价心理学（高价反而增信），后者是 B 端决策链的现实——决策人的个人利益往往是成交开关。
- <strong>你能用</strong>：报价要有锚和理由（贵在哪、为什么值），并搞清楚"谁拍板、他的利益是什么"。<strong>不要误读成</strong>——回扣属灰色操作、各地合规与商业道德风险自负，这里只作为"理解 B 端决策动机"的样本，不建议照抄。

### 5. "流量不是瓶颈，交付才是"——这类生意的真实卡点
反复出现的对话："你那边后端都接不过来吗？""不是接不过来，是都能接、没时间做。"他的流量"很棒"，一个闲鱼号、一个小红书号就接不完，所以他根本不想买流量；真正的卡点是后端交付产能（会做的人少、要筛、不然利润低）。
- <strong>为什么重要</strong>：很多人默认"生意难=缺流量"，但服务型生意常常反过来——缺的是产能/可信交付。这决定了你该投钱在获客还是在团队。
- <strong>你能用</strong>：诚实判断你的瓶颈在漏斗哪一段，别在不缺的环节继续砸钱。这也是他撮合"两个管前端、两个管后端"分工的根因。

### 6. 顺带露出的资源：论文代写 / 单片机供货（堆量打法）
- <strong>论文代写/SCI</strong>：他说自己能写 SCI 计算机 4/5 区，团队还能给博士写论文，"几千块包辅导"，熟图像分割/目标检测/轻量化；研究生群体便宜量大。这是文件名"AI写作资源"的真实出处。
- <strong>单片机供货</strong>：他有深圳、福建两个传统行业单片机合作方，"一个才两三百，你卖到 1000，就堆量"，可转介给想做这块的人；闲鱼对单片机流量"好像不限制"。
- <strong>为什么重要</strong>：这展示了一种"资源中介"收入——你自己不一定干，但<strong>手上的供给方/产能本身就能转介变现</strong>。
- <strong>你能用</strong>：盘点你认识的"能交付方"，撮合本身就是一门生意。<strong>不要误读成</strong>——论文代写涉及学术诚信与违规风险，此处仅作"资源中介模式"的结构样本，不做背书。

## 🛠 可直接复用的方法（含"不要误读成"）

<strong>方法 1：先判断瓶颈在"信任"还是"产能"</strong>
- 核心意思：to-小B 服务里，成交卡在信任、交付卡在产能，二者要分开治。
- 你可以怎么用：复盘最近 10 个未成交/未交付的单，归因到"不信我"还是"做不过来"，对症下药。
- 不要误读成：以为多买流量就能解决——他的案例恰恰是流量过剩、产能不足。

<strong>方法 2：用实物/线下做"信任放大器"</strong>
- 核心意思：把抽象能力变成客户当面能看、能摸、显然不简单的东西。
- 你可以怎么用：准备一个"演示道具/线下环节"作为成交临门一脚。
- 不要误读成：堆华丽 PPT——他靠的是"机器狗都能做"这种难度可感的实物。

<strong>方法 3：报价锚定 + 用"贵=靠谱"叙事</strong>
- 核心意思：成本 3 万报 4–5 万，并主动说"我们比市面贵但更靠谱"。
- 你可以怎么用：给高价配一个客户买得下去的理由（更稳、有案例、能负责）。
- 不要误读成：单纯涨价——锚没有"为什么值"的叙事支撑，只会吓跑客户。

<strong>方法 4：现金牛养现金、另押长期方向</strong>
- 核心意思：用代做（能立刻赚钱但会卷）养着，主押出海（工具站/游戏站/SaaS）。
- 你可以怎么用：明确区分你的"现金来源"和"未来下注"，别让短期单子吃光你押注的时间。
- 不要误读成：脚踏两只船=分心——前提是现金牛能半自动跑，才腾得出手押新方向。
